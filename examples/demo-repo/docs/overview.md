# Citybikes — project overview

Citybikes is a (fictional) bikeshare analytics project: ride exports land in `db/`,
queries live in `sql/`, exploration in `notebooks/`. The network runs **17 active
stations** and has recorded about **1,280 rides** to date (tracked — see
`librarian verify`, check `total_rides`).

Start with `docs/schema.md` for table shapes before writing any query.
